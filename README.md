# 🇸🇱 Salone Safe Ride - Sierra Leone National Transport Safety System

## Mission
To protect every passenger traveling across Sierra Leone's 16 districts by replacing insecure paper manifests with a secure digital system that enables real-time tracking, emergency SOS, and family communication.

## The Problem We Solve
- Paper manifests get lost or burned in accidents
- Families have no way to find loved ones after crashes
- Emergency responders have no passenger information
- Families forced to search on social media for victims

## Our Solution
✅ Digital Manifest - Secure cloud storage, never lost
✅ Live GPS Tracking - Family monitors journey in real-time
✅ One-Tap SOS - Alerts family, police, and hospitals
✅ Police Portal - Instant manifest access during emergencies
✅ Driver Rating System - Community-driven safety standards
✅ Accident Hotspot Map - Real-time danger zone alerts

## Technology
- Backend: PHP 8+
- Database: MySQL
- Frontend: HTML5, CSS3, JavaScript
- Maps: Leaflet.js / OpenStreetMap
- Server: Apache (XAMPP)

## Installation for Sierra Leone Transport Parks
1. Install XAMPP on Windows computer
2. Copy all files to `C:\xampp\htdocs\saloneride\`
3. Start Apache and MySQL services
4. Import `saloneride_db` database
5. Access at `http://localhost/saloneride/`

## User Roles
| Role | Access |
|------|--------|
| Passenger | Create trips, share with family, SOS |
| Driver | Confirm trips, update status |
| Park Manager | Create manifests for passengers |
| Police | Access manifests during emergencies |
| Admin | Full system management |

## Emergency Numbers (Sierra Leone)
- 🚓 Police: 119
- 🚑 Ambulance: 999
- 🔥 Fire Force: 998
- 🛣️ Road Safety: 112
